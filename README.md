# 国内优秀虚拟展厅案例调研

## Virtual Exhibition Hall Research

---

### 📖 项目简介

本项目系统梳理了国内 **15个** 顶尖虚拟展厅/数字展览案例，涵盖博物馆、档案馆、纪念馆、展览馆等多种机构类型。通过从展示形式、技术实现、内容组织、交互设计等多个维度进行深度分析与横向比较，提炼出可供参考的优秀实践启示。

**项目性质**：调研成果展示系统 / 高校课题汇报 / 项目验收展示

**设计水准**：Apple/腾讯/阿里云级别现代卡片式设计，达到作品集级别

### 🚀 运行方式

1. **直接打开**：双击 `index.html` 即可在浏览器中查看
2. **本地服务器**（推荐）：
   ```bash
   # Python 3
   python -m http.server 8080
   # 或使用 VS Code Live Server 插件
   ```
3. **部署到服务器**：将整个目录上传至任意静态文件服务器（Nginx / Apache / CDN）

**浏览器兼容**：Chrome 90+ / Edge 90+ / Safari 15+ / Firefox 90+

### ➕ 新增案例方法

1. 打开 `js/main.js`
2. 在 `casesData` 数组末尾添加新的案例对象：

```javascript
{
    id: 'case-16',                           // 唯一ID，格式 case-XX
    name: '新的案例名称',
    org: '所属单位',
    url: 'https://官方网站地址',
    format: '展示形式（VR全景/网页叙事型/3D展示等）',
    formatIcon: '🥽',                         // 展示形式图标
    type: '机构类型（博物馆/档案馆/纪念馆/展览馆）',
    typeIcon: '🏛️',                           // 机构类型图标
    difficulty: 2,                            // 制作难度 1=简单 2=中等 3=复杂
    recommend: 85,                            // 推荐指数 0-100
    summary: '案例简介，2-3句话描述。',
    features: [
        '特色分析要点1',
        '特色分析要点2',
        '特色分析要点3'
    ],
    worthLearning: [
        '值得借鉴的内容1',
        '值得借鉴的内容2',
        '值得借鉴的内容3'
    ],
    scenes: ['适用场景标签1', '适用场景标签2', '适用场景标签3'],
    screenshots: [
        { src: 'images/case16-01.jpg', alt: '截图1描述' },
        { src: 'images/case16-02.jpg', alt: '截图2描述' },
        { src: 'images/case16-03.jpg', alt: '截图3描述' }
    ]
}
```

3. 将截图文件放入 `images/` 目录
4. 刷新页面即可看到新案例

### 🖼️ 截图说明

**当前状态**：36张 `.jpg` 格式的网站真实截图（12个案例）+ 9张 `.svg` 占位图（3个案例因网站连接限制使用占位图，标注了"请手动替换"）。

**截图来源**：使用 Puppeteer 自动化工具对各案例官网进行截取，1280×800 视口。

**如需替换**：将新截图命名为 `caseXX-01.jpg`、`caseXX-02.jpg`、`caseXX-03.jpg` 放入 `images/` 目录覆盖原文件即可。
