# 国内优秀虚拟展厅案例调研

## Virtual Exhibition Hall Research

---

### 📖 项目简介

本项目精选国内 **5个** 优秀虚拟展厅/数字展览案例，按平面网页叙事型、全景扫描型、3D建模交互型、数据库/档案检索型四大分类组织。从展示形式、技术实现、内容组织、交互设计等维度进行深度分析与横向比较，提炼可供参考的优秀实践启示。

**项目性质**：调研成果展示系统 / 课题汇报 / 项目验收展示

### 🚀 运行方式

1. **直接打开**：双击 `index.html` 即可在浏览器中查看
2. **本地服务器**（推荐）：
   ```bash
   # Python 3
   python -m http.server 8080
   ```
3. **部署到服务器**：将整个目录上传至任意静态文件服务器

**浏览器兼容**：Chrome 90+ / Edge 90+ / Safari 15+ / Firefox 90+

### ➕ 新增案例方法

在 `js/main.js` 的 `casesDataRaw` 数组末尾添加案例对象：

```javascript
{
    id: 'case-XX',
    name: '案例名称',
    org: '所属单位',
    url: 'https://...',
    category: '平面网页叙事型',           // 四大分类之一
    categoryIcon: '🌐',
    format: '网页叙事型',
    formatIcon: '🌐',
    type: '博物馆',
    typeIcon: '🏛️',
    difficulty: 2,                       // 1=简单 2=中等 3=复杂
    recommend: 85,                       // 推荐指数 0-100
    summary: '案例简介。',
    features: ['特色1', '特色2', '特色3'],
    worthLearning: ['借鉴1', '借鉴2', '借鉴3'],
    scenes: ['场景标签1', '场景标签2'],
    screenshots: [
        { src: 'images/caseXX-01.jpg', alt: '截图描述' },
    ]
}
```

将截图放入 `images/` 目录，刷新页面即可。

**注意**：案例按 `category`（四大分类）分组排序，同一分类内按 `recommend` 降序排列。无截图的案例设置 `screenshots: []` 即可，页面会自动显示占位提示。
